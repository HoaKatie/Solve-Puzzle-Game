SOLVE THE PUZZLE GAME

The app will allow users to solve a puzzle by dragging words around on the canvas with their mouse. Upon completion, the text represented by the user's dragged words will need to be reconstructed and sent to the server for verification of the puzzle's accuracy.

To launchb the server, use command: node server.js

URL to visit: http://localhost:3000/index.html 

node.js version number:  v18.13.0.

Operating system: Windows

No external modules
